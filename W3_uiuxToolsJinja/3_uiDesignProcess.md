
# User Interface Design:

- Design for interaction with user

- Goals:
    - Simple - easy for user to understand and use
    - Efficient - user achieves goal with minimal effort

- Asthetics:
    - How good does something look
    - This is very difficult to teach
    - For most, this refer to "what is more pleasing
    - For beginners, it is recommended to use the design, ideas that are already tried & tested and are known to be visually pleasing
    - This is where bootstrap comes into the picture

- Accessibility
    - How accessible/usable is the website is for someone who has some kind of disability or imparement
    - Asthetics != Accessible


# Systematic process
    - Functionality requirement gathering - what is needed?
    - User and Task analysis - user preference, task needs
    - Prototyping - wireframes, mockups
    - Testing - user acceptance, usability, accessibility
